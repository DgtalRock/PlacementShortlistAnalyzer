# Placements Analyzer
Extract rollno from PDFs to display stats.

## Getting Started

#### Activate virtual environment
```
. venv/bin/activate
```

#### Run command
```
python main.py --file <file_name>
```
Example: 
```
python main.py --file test.pdf
```
#### Add watch rolls
Add your friends roll numbers to "friends.txt" for easy tracking.
